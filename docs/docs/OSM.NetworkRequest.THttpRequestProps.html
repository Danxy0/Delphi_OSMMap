Description

Fields

Proxy: string;

[http://]host:port Address of HTTP (CONNECT) proxy, protocol is optional.
Direct connection if empty. If equals to SystemProxy, OS-wide value is used.
Could contain login in the form user:pass@host:port.

URL: string;

HTTP URL to access. Could contain login in the form proto://user:pass@url overriding HttpUserName and HttpPassword fields.

HttpUserName: string;

Access login name.

HttpPassword: string;

Access login pass.

HeaderLines: TStrings;

HTTP header.

Any data that request function could use
